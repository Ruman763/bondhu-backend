## Features Bondhu does NOT have (like WhatsApp)

| Feature | Description |
|--------|-------------|
| **Read receipts** | “Seen” / blue ticks when the other person has read the message. Bondhu does not show read state. |
| **Disappearing messages** | Auto-delete messages after 24h / 7d / 90d. Not present in Bondhu. |
| **Forward message** | Forward a message to another chat (with or without “forwarded” label). Bondhu has no forward action. |
| **Broadcast list** | Send one message to many contacts without a group. Not in Bondhu. |
| **Starred / saved messages** | Pin important messages in a “Starred” list. Bondhu has no starred messages. |
| **Archive chat** | Hide chat from main list but keep it (restore later). Bondhu has clear/delete, not archive. |
| **Edit message** | Edit sent text after sending. Bondhu has delete only, no edit. |
| **Polls** | Create in-chat polls (e.g. “Where to eat?”). Not in Bondhu. |
| **Share live location** | Share real-time location for 15 min / 1 hour / 8 hours. Bondhu has profile “location” text only. |
| **Document preview** | In-chat preview for PDFs/docs (not only “file” + open externally). Bondhu opens files externally. |
| **Chat backup / restore** | Backup chats to Google Drive / iCloud and restore on new device. Bondhu has local storage, no cloud backup. |
| **Linked devices / multi-device** | Use same account on phone + tablet/web at once. Bondhu is single-session. |
| **WhatsApp Business** | Business profile, catalog, quick replies, away message. Not in Bondhu. |
| **QR code add contact** | Add contact by scanning QR. Bondhu has search/contacts only. |
| **In-app payment (WhatsApp Pay)** | Send/receive money in chat (in supported regions). Bondhu has wallet UI but no real payment. |

## Features Bondhu does NOT have (like Facebook / Messenger)

| Feature | Description |
|--------|-------------|
| **News Feed algorithm** | Chronological vs “Most relevant” feed with ranking. Bondhu feed is chronological. |
| **Share to Story** | Share a post or external link to your story. Not in Bondhu. |
| **Share post** | Share someone’s post to your timeline or to a friend. Not in Bondhu. |
| **Save to collection** | Save posts to named collections (e.g. “Recipes”). Bondhu has “save” only, no collections. |
| **Stories archive / Highlights** | Save stories to profile “Highlights” (permanent). Bondhu stories are ephemeral, no highlights. |
| **Reels / short video feed** | Dedicated short-form video feed (like Reels/TikTok). Bondhu has a “Videos”/Reels tab on profile but no full Reels feed. |
| **Live video** | Go live and stream to followers. Not in Bondhu. |
| **Rooms / video rooms** | Multi-person video rooms. Bondhu has 1:1 video only. |
| **Marketplace** | Buy/sell items. Not in Bondhu. |
| **Events** | Create and manage events, invite, RSVP. Not in Bondhu. |
| **Pages** | Public pages for brands/creators. Not in Bondhu. |
| **Rich groups** | Group roles (admin, moderator), rules, approval, announcements. Bondhu has basic groups only. |
| **Photo/video albums** | Profile albums (e.g. “Photos”, “Videos”). Bondhu has grid of posts, no albums. |
| **Check-in / place tag** | Tag location on a post. Bondhu has profile “location” text only. |
| **Tag people in post** | @mention in post that notifies and links to profile. Not in Bondhu. |
| **Memories / On This Day** | “On this day” or memories from past years. Not in Bondhu. |
| **Birthday reminders** | Show friends’ birthdays. Not in Bondhu. |
| **Dating** | Dating profile and matches. Not in Bondhu. |
| **Crisis / safety** | Safety check, crisis response. Not in Bondhu. |
| **Ads / business tools** | Ad platform, Insights, Business Suite. Not in Bondhu. |
| **Payments (FB Pay)** | Send money in Messenger / Facebook. Bondhu has wallet UI only. |

## Summary: high-impact gaps (good roadmap candidates)

**Chat (WhatsApp-like)**  
- Read receipts  
- Disappearing messages  
- Edit message  
- Forward message  
- Starred messages  
- Archive chat  
- Polls  
- Share live location  
- Chat backup/restore  

**Social (Facebook-like)**  
- Share to Story  
- Share post  
- Story Highlights  
- Full Reels/short-video feed  
- Live video  
- Rich groups (roles, rules)  
- Marketplace or Events (if you want that positioning)  

**Common**  
- In-app payments (real integration, not just UI)  
- Multi-device / linked devices  

You can use this list to decide what to build next based on user demand and backend complexity.
